# Middleware

| Endpoint        | Params                               | Service |
| :-------------- | :----------------------------------- | :------ |
| `/autocomplete` | `q={search}`, `sessionToken={token}` | [Place autocomplete](https://developers.google.com/places/web-service/autocomplete) |
| `/details`      | `q={search}`                         | [Place details](https://developers.google.com/places/web-service/details) |

## Usage

```js
import express from 'express'
import googleMapsMiddleware from '@emcasa/places-autocomplete/middleware'

const server = express()

server.use('/maps', googleMapsMiddleware({
  apiKey: '...',
  // ... @google/maps client options
}))

```
